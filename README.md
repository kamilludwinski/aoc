# aoc

This repo contains my Advent of Code solutions.

# Rationale of why AOC

Every year (starting from 2015) December brings us (programmers) a gift - Advent.
No, not the one you think about, Advent of Code.
Advent of Code is a set of challenges, allowing us to experiment with new languages.
